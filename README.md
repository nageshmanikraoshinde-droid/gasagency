 # GasEase – LPG Gas Cylinder Booking System

GasEase is a lightweight web-based LPG Gas Cylinder Booking System developed as an academic project. It enables users to register, log in, book LPG gas cylinders, make payments, manage their profiles, and generate booking receipts through a simple and responsive interface.

---

## 📌 Project Description

The main objective of GasEase is to simplify the LPG gas cylinder booking process by providing an easy-to-use online platform. Customers can create an account, book gas cylinders, complete payments, and receive digital receipts without visiting the gas agency.

---

## ✨ Features

- Home Page
- User Registration
- User Login & Logout
- User Profile Management
- LPG Cylinder Booking
- Online Payment
- Receipt Generation (PDF)
- Responsive User Interface
- Secure Authentication

## 🚀 Installation

1. Install XAMPP.
2. Copy the project folder into the `htdocs` directory.
3. Start Apache and MySQL from the XAMPP Control Panel.
4. Create a MySQL database named `gasease`.
5. Import the SQL database file.
6. Open your browser and visit:

```
http://localhost/GasEase/
```

## 📋 Project Workflow

1. Open the Home Page.
2. Create a new account.
3. Log in with your credentials.
4. Book an LPG gas cylinder.
5. Complete the payment process.
6. Generate and download the booking receipt.
7. View or update your profile.
